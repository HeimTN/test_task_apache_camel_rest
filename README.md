# Управление вакансиями и подпиской на уведомления

Этот проект представляет собой веб-сервис, созданный для компании "Цифровое Будущее". С помощью этого сервиса вы можете легко управлять вакансиями, искать подходящие предложения и подписываться на уведомления о новых вакансиях.

## Описание

Веб-служба предоставляет REST API для выполнения следующих задач:

- **Загрузка новых вакансий**: Вы можете добавлять новые вакансии в систему.
- **Поиск вакансий**: Вы можете осуществлять поиск вакансий по различным параметрам.
- **Подписка на уведомления**: Пользователи могут подписываться на получение уведомлений о новых вакансиях, соответствующих их интересам.

## Информация о вакансии

Каждая вакансия содержит следующие атрибуты:

1. **Наименование**: Название вакансии.
2. **Описание**: Описание вакансии.
3. **Должность**: Название должности.
4. **Уровень зарплаты**: Ожидаемый уровень зарплаты.
5. **Требуемый опыт работы**: Требуемый опыт работы для данной вакансии.
6. **Город**: Город, где расположена вакансия.

## Информация о соискателе

При подписке на уведомления пользователи предоставляют следующие данные:

1. **E-mail**: Адрес электронной почты пользователя.
2. **ФИО**: Полное имя пользователя.
3. **Город**: Город проживания пользователя.
4. **Интересующая должность**: Должность, в которой пользователь заинтересован.

## Методы веб-службы

1. **PUT /vacancy**: Создание новой вакансии.
2. **GET /vacancy**: Получение списка вакансий.

   *Необязательные параметры:*
   - `name`: Фильтр по наименованию вакансии.
   - `position`: Фильтр по названию должности.
   - `city`: Фильтр по городу.

3. **PUT /subscriber**: Добавление нового подписчика.

## Отправка уведомлений

Раз в сутки пользователи получают уведомления по электронной почте о новых вакансиях, соответствующих их предпочтениям.

Пример уведомления:

```
Здравствуйте, [ФИО]!

Информируем вас о новой вакансии на должность "[Должность]".
Наименование: [Наименование]
Описание: [Описание]
Уровень зарплаты: [Уровень зарплаты]
Требуемый опыт работы: [Требуемый опыт работы]

С уважением,
Цифровое Будущее
[дата]
```

## Технические требования

- **Spring Boot 2**: Для разработки веб-службы.
- **Apache Camel**: Для реализации рассылки уведомлений.
- **HSQLDB**: Для хранения данных.
